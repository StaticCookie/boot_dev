Terminal Commands

Directory commands
mkdir {name} | Make directory / folder
ls | List Directories
cd {directory name} | Changes directory (~ for route)
touch {filename.extension} | Create File


Commands related to a remote repository:
git clone git@github.com:USER-NAME/REPOSITORY-NAME.git
git push or git push origin main (Both accomplish the same goal in this context)

Commands related to the workflow:
git add .
git commit -m "A message describing what you have done to make this snapshot different"

Commands related to checking status or log history
git status
git log
git log [<remote>/<branch>] (Show the log of a remote branch).

The basic Git syntax is:
program | action | destination.

Create a new Branch: 
git branch {branch_name}        (Creates the branch)
git switch -c {branch_name}     (Creates and switches to the branch)

Show the Branch you are on:
git branch

Show all branches:
git log --oneline --graph --all
git log --oneline --decorate --graph --parents

Switch branches:
git switch {branch}              (Recommended) 
git checkout {branch}            (Old method)

Merge Branches: (from Main)
git merge {branch}

Delete Branch: 
git branch -d {branch}

Rename Branch: 
git branch -m oldname newname

Rebase Git Branch
git rebase {banch}
git rebase main                 (to rebase from main)

Git Reset 
(Soft)
git reset --soft COMMITHASH

Git merge 
git remote add <name> <uri>
git merge [<remote>/<branch>]

Git fetch 
find .git/objects
git fetch <remote> <branch>
